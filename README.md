#👕 E-Commerce Clothing Store

A fully functional E-commerce web application for clothing, built using **Python** and **Django** backend.

## 🚀 Features
* **User Authentication:** Sign up, login, and logout.
* **Product Catalog:** Browse clothing items by category.
* **Shopping Cart:** Add, update, and remove items from the cart.
* **Secure Checkout:** Smooth order processing.

## 🛠️ Technologies Used
* **Backend:** Python, Django Web Framework
* **Database:** SQLite (default)
* **Frontend:** HTML, CSS, Bootstrap
